This directory should contain the following distributions:

lamarc-2.1.7.Linux-i686-gtk.tar.gz

x86_64 linux executables (invoked from command line)

We have tried to make these as generically applicable as possible, but given the wide variation in Linux setups, it is possible that you will need to compile for your specific Linux machine.

lamarc-2.1.7-osx.dmg

OS X clickable executables; tested on 10.5.8, 10.6.8, and 10.7.3.

lamarc-2.1.7-win32.zip

32-bit clickable executables.

lamarc-2.1.7-win64.zip

32-bit clickable executable for the lam_conv file converter;
64-bit clickable executable for the main LAMARC program.

lamarc-2.1.7.src.tar.gz

Full LAMARC distribution. Use this if you need to compile LAMARC.

lamarc-2.1.7.doc.tar.gz

Documentation only. This is included in each of the other distributions, so you probably don't need it.
